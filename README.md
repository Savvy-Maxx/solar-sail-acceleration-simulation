# Optimizing Solar Sail Acceleration Using Reflectivity Variations

## Overview
This repository contains code and figures for an independent physics research project investigating how reflectivity affects solar sail acceleration.

## Structure
- `paper/` — Research paper PDF
- `src/` — Python simulation code
- `figures/` — Generated plots
- `data/` — Sample outputs
